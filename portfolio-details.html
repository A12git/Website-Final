Activity 101 Free test (Draw)

In this Activity we are asked to draw a character or a logo. I drew a character in a game that I play called Punishing Gray Raven the name is Kuroro.

Activity 102 (Shapes and Forms)

We are asked to Draw various shapes and forms.

Laboratory Act 101 (Draw a Sleeping Fox)

Drawing a Sleeping Fox in Adobe Illustrator using different shapes, colors and values with a template.

Laboratory Act 102 (Draw a Awake Fox)

Using the Shape builder tool we drew the awake fox by combining and removing shapes.

Laboratory Act 103 (Draw a Impossible shape)

Using the Line Segment Tool I made 3 identical horizontal lines then copy pasted it then rotated it to 120 and repeated it again but this time rotated it to 240 using those lines I made a triangle then used the Shape Builder Tool to separate the shapes I then used the color green then changed the value to the different shapes to create a Impossible Shape.

Laboratory Act 104 (Draw with the Curvature Tool (Owls Oh My!))

Using the Curvature tool I drew 4 different characters.
